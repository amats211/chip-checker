About the Grade System

The WITH CODE chips are graded A- to A++++ based on chip generation, interface (eMMC 4.5 / 5.1 / UFS 2.1 / 3.1), and board spec configuration shown in 规格 (guige) column of the reference charts.

NON-CODE chips do not appear in these lists. They are priced by storage capacity only (8GB / 16GB / 32GB / 64GB / 128GB / 256GB / 512GB).

How to Use

  • Tap Camera or Gallery to load a chip photo
  • Or type the model number manually
  • Graded chips show grade, type, storage and price
  • Unknown chips show as possible non-code
  • Select GB size to save it to the Non-Code DB
  • Or tap Search Online to look it up in your browser

Grade Scale

A++++ — UFS 3.1 · Pinakamataas
A+++ — UFS 2.1/2.2 · High-end
A++ — eMMC 5.1 · Premium
A+ — eMMC 5.1 · Standard
A- — eMMC 4.5 · Older gen
Non-Code — No grade, priced by GB
