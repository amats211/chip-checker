Grade System Explained

WITH CODE chips are graded A1–A5 based on chip generation, memory interface (eMMC 4.5 / 5.1 / UFS 2.1 / 3.1), and board configuration shown in the 规格 (guige/specs) column of the reference charts above.

NON-CODE chips do not appear in the grade lists. They are priced by storage capacity only (8 / 16 / 32 / 64 / 128 / 256 / 512 GB).

How to Use

  • Tap Camera or Gallery to load a chip photo
  • Read the marking and type the model number
  • Graded chips show A1–A5 grade, type, storage and price
  • Unknown chips show as possible non-code
  • Select GB size to save to Non-Code database
  • Tap Search Online to look up chip in browser
  • Add buyers with their own price lists in Buyers tab

Grade Scale (A1–A5)

A5 (A++++) — UFS 3.1 · Pinakamataas
A4 (A+++) — UFS 2.1/2.2 · High-end
A3 (A++) — eMMC 5.1 · Premium
A2 (A+) — eMMC 5.1 · Standard
A1 (A-) — eMMC 4.5 · Older gen
Non-Code — No grade, by GB only
