Grade System (A1–A5)

Grades are based on EMCP storage capacity classification used in the PH repair market:

A5 = 256GB (A++++ / UFS 3.1)
A4 = 128GB (A+++ / UFS 2.x combo)
A3 = 64GB (A++ / eMMC 5.1 high-end)
A2 = 32GB (A+ / eMMC 5.1 standard)
A1 = 16GB (A- / eMMC 5.1 entry)
Non-Code = No grade, priced by GB

How to Use

  • Tap Camera or Gallery to load a chip photo
  • Read the marking and type the model number
  • Matched chips show grade A1–A5, storage, type and price
  • Unknown chips show as possible non-code
  • Select GB size to save it to Non-Code database
  • Tap Search Online to look up chip in browser
  • Add buyers with their own price lists in Buyers tab
